## 🚀 Overview

A simple cybersecurity tool that analyzes password strength and provides actionable suggestions to improve it.

This project demonstrates **basic security practices**, including password validation and pattern detection using Python.

---

## ✨ Features

✔️ Checks password length
✔️ Detects uppercase & lowercase characters
✔️ Verifies numbers and special symbols
✔️ Provides strength rating:

* 🔴 Weak
* 🟡 Medium
* 🟢 Strong

✔️ Gives improvement suggestions

---

## 🧠 How It Works

The program evaluates passwords based on 5 key criteria:

* Minimum length (8+ characters)
* Uppercase letters (A–Z)
* Lowercase letters (a–z)
* Numbers (0–9)
* Special characters (!@#$...)

Each condition increases the **security score**.
